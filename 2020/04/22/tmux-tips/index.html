tmux中使用vim快捷键

修改.tmux.conf:

  • 配置vim的方向键hjkl
  • 来切换panel C + b + [h/j/k/l]
  • 分屏与新建窗口:
    C + b + “: 上下分屏
    C + b + %: 左右分屏
    C + b + c: 新建窗口

我的工具箱

发表于 2016-02-22 分类于 工具 Disqus：

自认为是个工具控, 还记得小的时候，看动画片水果城中的香蕉总是能在非常时刻从他的背包中找到需要的工具，从此心灵深处埋了颗种子, 上大学时总喜欢把自己所需要的东西背在书包里面，现在工作了， 今天终于可以把这2年使用的应用做个汇总啦.

系统

开发服务器使用centos, 个人最开始是2010年使用ubuntu, 之后由于ubuntu系统升级后桌面环境由gnome变为了unity，之后我的电脑变慢了，于是搜索了大牛linus torvalds使用的是轻量级的xfce桌面环境，于是我也进来了。去年发现linux mint 发行系列内置就是xfce，使用感觉很棒。最终推荐linux用户使用

  • Linux Mint

编辑器

  • vim,emacs
    编辑神器自不用说，不过自己主力用vim. emacs之前由于org的功能，使用过一段时间，由于中文操作与美感不太满意，
    就没有坚持使用的意义，而且由于自从2年前vim与插件做为我的唯一IDE开发环境后，
    已习惯的jkl.
    • vim IDE 配置，记得2年前我也是通过这个入门的。

      vim 推荐插件:

      • bundle来管理插件
      • cscope tag跳转插件

笔记类

  • evernote
    • 微信,浏览器插件保存
    • chrome浏览器中搜索，可以搜索笔记内容
    • 类似的插件很多如:
      • [clearly] 去广告阅读时使用
      • [evernote web clipper] 保存网页到evernote

  • wiznote是国内做的也很不错的一个我们团队文件共享使用这个。

  • cherrytree
    最开始了解渠道来自巴蛮子
    喜欢的特点:

    • 树状笔记列表
    • F8生成当天结点记笔记
    • 可内嵌图片与代码
      同时与坚果云结合使用来弥补cherrytree不能网络同步备份文件的不足

浏览器与插件

  • 书签管理
    使用chrome 或者firefox 账号来同步书签，如果你像我一样同时使用chrome与firefox时，可以使用xmarks插件来同步书签。
  • 密码管理
    firefox与chrome都可以保存一些不太重要的密码，但firefox可能更安全一些:可以设置master密码, 每次查看或者启动时需要输入master 密码.
    当然也试用过lastpass但感觉使用体验并不怎么舒服，就停了下来，到是很期待无密码的时代.
  • 阅读同步类
    • pocket
    • evernote
    • send2kindle

阅读类

  • pocket
    支持手机与浏览器插件，把你来不及看的网页放入pocket里面，回头再看。
  • calibe 管理电子书的开源软件
    用于管理kindle资源真心很棒，也能转换电子书格式
  • RSS 阅读器
    目前使用feedly, 关于订阅内容下次需要单独抽出来记录一下。

搜索引擎

  • google
  • shodan
    国内也有个类似的zoomeye
  • 图标搜索
    • iconpng
    • findicons
    • iconfinder

扫描端口使用，上次redis安全漏洞, 使用这个扫描redis端口，进了好些服务器:)

云盘

  • 坚果云
    之前使用的是dropbox，自从被墙后，找了好多linux平台的同步工具都不理想，直到遇到她(坚果云)
    支持文件版本回滚, 而且同步速度真的很快，和dropbox使用体验几乎没有差别。
  • 百度云
    主要用于离线下载电影或者大型文件，用于加速:)

输入法

  • rime
    感觉五笔输入法这是感觉最舒服的一款。

GFW

之前使用了红杏的插件，goagent等现在都不行了，目前使用公司的亚马逊云+chrome插件SwitchyOmega+ 本地ssh代理来科学上网.
SwitchyOmega中配置映射端口与国内的GFWlist列表, 然后就可以了。如果在windows下面, 也可以使用putty.exe 监听本地端口做代理. windows下请参考(http://www.extremetech.com/computing/93106-escaping-the-firewall-with-an-ssh-tunnel-socks-proxy-and-putty)

  • 开启本地代理

  • 其它软件使用代理

github tsocks or tsocks.sourceforge

画图

  • xmind
    上大学时，看了思维导图这本书，之前好奇使用过一段时间，不过这种思维导图的思路已经在头脑中形成了，不再局限于形式，目前更喜欢用彩笔在纸上画。
  • js 序列图或流程图
    目前好多markdown编辑器中已经支持这画图(为知笔记已经支持)，对于写程序设计类文档来太方便了，感谢开源。

    • 作业部落
